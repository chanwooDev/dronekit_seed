### 요구사항

땅을 지정할 수 있다.
- 땅은 직사각형을 기반으로 한다
- 현재는 json 파일 기반으로 수행함

드론은 특정 땅 안에서 특정 각도로 서보모터를 세팅한다

- 드론의 현재 위치를 파악한다
- 드론의 위치에 따른 서보모터를 조정한다
    
    - [ ] 특정 구역에 들어온 첫 시점에 명령을 한번 내린다
    - 구역의 변경은 짧은 시간 주기로 반복해서 확인한다
        - 구역이 변경된 첫 순간만 변경 신호를 보낸다
    - 서보모터 각도를 조정한다
    - 특정 땅의 기준이 되는 수치는? -> 위도 경도
